Weather data project

This is a preview of the data collected using my weather data project. It grabs data from Accuweather using the provided API and LOCATION key, then saves it to a json to be displayed on the webpage. I ran this on my homelab from June-September 2025. There are a few missing bits of data due to the script quitting itself sometimes.
