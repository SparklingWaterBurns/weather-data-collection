# weather-data-collection

A set of python scripts that collect data from AccuWeather API and save it to a .JSON file that gets read by chart.js in the webpage used for displaying the chart.

This includes data i've collected for my town in June-September 2025.

# How it works
Gets data using your api and location key from accuweather.
Saves it to a JSON file.
The webpage displays the data using chart.js
